# Web Standards Days

## Установка и запуск дев-окружения

Необходимо наличие NodeJS версии не ниже 4.1. Допускается использование nvm.

1. `npm i` — устатновка зависимостей
2. `npm run server` - запуск дев-сервера и вотчера изменений
