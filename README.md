# Web Standards Days

## Установка и запуск окружения

Необходима Node.js версии не ниже 4.1.

1. Установка зависимостей: `npm i`.
2. Локальный серверс с вотчером: `npm run server` или `gulp` (если Gulp установлен глобально).
